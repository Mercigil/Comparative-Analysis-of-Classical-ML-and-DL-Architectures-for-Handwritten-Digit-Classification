# Comparative-Analysis-of-Classical-ML-and-DL-Architectures-for-Handwritten-Digit-Classification
Handwritten digit recognition remains a core problem in computer vision, with both classical machine learning (ML) and deep learning (DL) models widely applied but rarely compared under identical conditions.
br
In this repository contains a research article "Comparative Analysis of Classical Machine Learning and Deep Learning Architectures for Handwritten Digit Classification" written by Tamakloe A. Vivian, Maikusa S. Abdulrahman, and Eli A. Jiya and published by Physics Access.
Also included is the excutable code implementation of the research, and a code & output showing the output to each implemtation.
